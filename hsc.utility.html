distance(x: array) → array¶

Calculates the distance between points in x and the bounding box.

The distance is calculated to the closest point on the box. Points inside the box will be returned as zero as their distance to the box is zero. The Frobenius norm is used to calculate the distance.

Parameters:

x – (n_dim, n) array of coordinates.

Returns:

array of distances.

inside(x: array) → ndarray¶

Method to tell which points from the input array are inside the bounding box.

Parameters:

x – (n_dim, n) array that should be probed.

Returns:

indices to all values in the array which are inside the bounding box.
