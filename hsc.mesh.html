hsc.mesh.crystal_builder module¶

class hsc.mesh.crystal_builder.CrystalBuilder(description: Description)¶

Bases: GmshBuilder

Base class for crystal builders.

Crystal builders are used to define all shapes of the crystals placed within a simulation domain. This base class does not place any crystals in the domain.

build() → List[int]¶

Builds the crystals according to the domain description.

Returns:

indices to all newly created crystals.

cut(objs: List[int], tools: List[int]) → List[int]¶

Cuts the tools from the objects.

Parameters:

  • objs – basic shape of the crystals.

  • tools – tools which are removed from the basic shape to create the crystal.

Returns:

indices to all crystals.

define_objects() → List[int]¶

Defines the objects associated with constructing the crystal by cutting.

Returns:

indices to all objects.

define_tools() → List[int]¶

Defines the tools associated with constructing the crystal by cutting.

Returns:

indices to all tools.

C-shaped crystal builder.

C-shaped sonic crystals use cylindrical shaped crystal as a precursor. The difference to a cylindrical shaped crystal is that a slot and a disk are cut from the disk.
