Holds information about absorbers.

hsc.domain_properties.domain_description module¶

class hsc.domain_properties.domain_description.Description(frequencies: ~numpy.array, max_frequency: float, rho: float, c: float, n_left: float, n_right: float, elements_per_lambda: float, elements_per_lambda_surf: float, absorber: ~hsc.domain_properties.absorber_description.AbsorberDescription, crystal: ~hsc.domain_properties.crystal_description.CrystalDescription, indices: dict = <factory>, processed: bool = False)¶

Bases: object

Holds information of a domain on which only the frequency is varied.

The reason for only varying the frequency in a single sample is that it may be easier to generate multiple samples on the same mesh. The frequency can but does not need to change the mesh of the domain. Compared to changing the parametrization of the crystals, this may save storage space. If the domain should be resampled for every simulation for every frequency, create one Description per frequency.

The C-shaped crystal is described by three individual properties. Each description is taken and arranged in a meshgrid to sample every possible permutation of parameters.
